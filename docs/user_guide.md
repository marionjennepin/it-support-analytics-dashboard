\# Guide utilisateur



\## Utilisation des données



Les fichiers générés peuvent être ouverts avec Excel ou Power BI.



Ils permettent d’analyser :

\- le volume de tickets

\- la performance des équipes

\- les catégories les plus problématiques





\## Fichiers disponibles



\- kpi\_summary.csv : indicateurs globaux

\- team\_performance.csv : performance par équipe

\- category\_performance.csv : analyse par catégorie

\- daily\_ticket\_volume.csv : évolution du volume de tickets
